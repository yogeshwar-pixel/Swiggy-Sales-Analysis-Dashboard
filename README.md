# Swiggy Sales Analysis Dashboard

## 📌 Project Overview
The **Swiggy Sales Analysis Dashboard** is an interactive Microsoft Excel dashboard designed to analyze food delivery sales performance across different cities and states in India. It provides meaningful business insights through dynamic charts, PivotTables, PivotCharts, slicers, and KPIs, allowing users to monitor sales trends, customer ratings, and regional performance.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor monthly sales trends.
- Compare sales across states and cities.
- Evaluate customer ratings.
- Track order volume and average order value.
- Create an interactive dashboard for business decision-making.

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)
- **Total Sales:** ₹53.01M
- **Average Rating:** 4.34
- **Average Sales per Order:** ₹268.51
- **Rating Count:** 5.59M
- **Total Orders:** 197.43K

### Dashboard Visualizations
- 📈 Monthly Sales Trend (Line Chart)
- 🍕 Sales by Food Category (Donut Chart)
- 🗺️ State-wise Sales Distribution (Filled Map)
- 📊 Monthly Sales Comparison (Column Chart)
- 🏙️ Top 5 Cities by Sales (Horizontal Bar Chart)
- 📅 Daily Sales Analysis (Column Chart)
- 🎨 KPI Summary Table with Conditional Formatting

### Interactive Filters
- Month
- Restaurant
- Category
- Order Type
- Payment Method
- Delivery Partner

## 🚀 How to Use
1. Download or clone this repository.
2. Open **Swiggy Dashboard.xlsx** using Microsoft Excel (2019 or Microsoft 365 recommended).
3. Navigate to the **Dashboard** worksheet.
4. Use the slicers to filter data by month, restaurant, category, or other available dimensions.
5. Analyze the KPIs and charts for business insights.
