Swiggy Sales Analysis Dashboard
📌 Project Overview

The Swiggy Sales Analysis Dashboard is an interactive Microsoft Excel dashboard developed to analyze food delivery sales performance across different cities and states in India. It provides business insights through dynamic charts, PivotTables, PivotCharts, slicers, and KPIs, enabling users to monitor sales trends, customer ratings, and regional performance.

🎯 Objectives
Analyze overall sales performance.
Monitor monthly sales trends.
Compare sales across states and cities.
Evaluate customer ratings.
Track order volume and average order value.
Provide an interactive dashboard for business decision-making.
